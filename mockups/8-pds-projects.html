Mockup 8 — PDS Projects: User Menu, Sidebar Save, Project Picker

What this mockup shows

  • Scene 1: Logged-out header (no change from current)
  • Scene 2: Logged-in header — username is a dropdown trigger with "My Projects" and "Log out"
  • Scene 3: Wide layout — sidebar with save button at bottom + save states
  • Scene 4: Narrow layout — accordion with save button below + save states
  • Scene 5: Project picker dialog (shown on login or via "My Projects")
  • Scene 6: Project picker with unsaved changes warning
  • Scene 7: Delete confirmation (type name to confirm)
  • Scene 8: Empty state (no saved projects)

Design decisions

  • Username click toggles a dropdown menu (same behavior on all breakpoints)
  • Save button lives at bottom of sidebar (wide) / below accordions (narrow)
  • Auto-save triggers on panel switch and card create/update, with brief "Saved" indicator
  • Manual save via the sidebar/accordion save button
  • "Back to intro" button removed
